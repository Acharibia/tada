Toast Types

Try different toast notification styles 

tada.success("Operation successful!");
tada.error("Something went wrong!");
tada.warning("Be careful!");
tada.info("Did you know?");

Positions

Choose where notifications appear

tada.show("Message", { position: "top-left" });

Advanced Features

Customize duration, animations, and behavior 

tada.show("Message", { duration: 10000 });
tada.show("Message", { duration: 0 });
tada.show("Message", { animation: { ... } });

Icons, Images & Progress

Add icons, images, and progress bars to your toasts 

// With icon
tada.show("Message", {
  icon: '<svg>...</svg>'
});

// With image
tada.info("New message from John", {
  image: "https://example.com/avatar.jpg"
});

// With progress
tada.show("Uploading...", {
  showProgress: true,
  duration: 5000
});

Border Radius

Customize the border radius for different styles 

// Pill shape
tada.success("Rounded!", {
  borderRadius: "50px"
});

// Square corners
tada.info("Sharp edges", {
  borderRadius: "0"
});

// Custom radius
tada.show("Custom", {
  borderRadius: "8px"
});

Custom Colors

Set your own background and text colors 

// Dark theme
tada.show("Dark mode!", {
  backgroundColor: "#1f2937",
  color: "#f3f4f6"
});

// Neon style
tada.success("Neon!", {
  backgroundColor: "#000",
  color: "#0ff"
});

// Custom gradient effect (use rgba)
tada.info("Ocean blue", {
  backgroundColor: "rgba(59, 130, 246, 0.9)",
  color: "#fff"
});
